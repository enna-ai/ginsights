# Git Track
Check who isn't following you back on Github using this simple go program

## Get Started

1. Edit `.env` file
```
GITHUB_TOKEN=your-github-token
GITHUB_USERNAME=your-github-username
```

2. Build executable
```go build```

3. Run the program
```go run .```
